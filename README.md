## 🚀 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/khaled1267/QurbaniHat.git

# 2. Navigate to project folder
cd QurbaniHat

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open **[http://localhost:3000](http://localhost:3000)** in your browser.

> ⚠️ **Note:** You may need to set up `.env.local` with your authentication credentials for `better-auth` to work properly.
